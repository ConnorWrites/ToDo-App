Conrad's Todo App (Full Stack)

A simple Todo application built with Node.js, Express, and MongoDB.  
This project shows basic CRUD operations, REST APIs, and database persistence using MongoDB Atlas.

Features:
- Add new todos
- View all todos
- Toggle todo completion
- Delete todos
- Storage with MongoDB
- Simple frontend using HTML, CSS, and JavaScript

Tech Stack:
# Backend
- Node.js
- Express
- MongoDB Atlas
- Mongoose

# Frontend
- HTML
- CSS
- JavaScript (Fetch API)
